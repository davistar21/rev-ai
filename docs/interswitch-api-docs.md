Purpose: Quick reference for understanding Interswitch's API offerings

WHAT ARE THESE APIs?
Interswitch provides APIs (Application Programming Interfaces) that allow developers to integrate payment, financial, and digital services into their applications without building everything from scratch.
In simple terms: Pre-built tools that let you add payment processing, bill payments, money transfers, and more to your app, just like plugging in a ready-made solution.

WHO USES THESE APIs?
Developers building apps that need payment, value added services and identity verification functionality
Startups creating fintech products
Businesses wanting to accept payments online or in-store
Banks and fintechs needing digital services for their customers

API CATEGORIES (9 MAIN GROUPS)

1. ACCEPT PAYMENTS
   Help customers pay you online or in-store. Details here and documentation here
   What you can do:
   Accept card payments (Verve, Visa)
   Create web checkout pages (customers pay on your website)
   Process Google Pay transactions
   Accept payments via POS (Point of Sale) devices
   Set up recurring payments (subscriptions)
   Generate payment links
   Process refunds
   Key APIs:
   Web Checkout API
   Card Payments API
   Google Pay API
   Recurring Payments API
   Payment Links API
   Refunds API

2. VALUE ADDED SERVICES (VAS)
   Let customers pay bills, buy airtime, and data. Documentation here.
   What you can do:
   Process bill payments (electricity, cable TV, water)
   Sell airtime and data bundles
   Validate customer details before payment
   Key APIs:
   Bills Payment API
   Airtime Recharge API
   Customer Validation API

3. SEND MONEY (TRANSFERS) 💸
   Move money between bank accounts or wallets. Documentation
   What you can do:
   Transfer money to bank accounts
   Retrieve the status of a transaction and validate account number
   Get list of Nigerian banks and their codes

4. LENDING & DATA SERVICES 💰
   Provide loans and access customer financial insights. Documentation.
   What you can do:
   Offer loans (small, short-term loans)
   Loan collection /repayment (manual and automatic)
   Data service and credit scoring
   Lending services - nano loans, value financing (customers can make payment for services on credit), salary lending.

5. CARDLESS SERVICES
   Let customers withdraw cash or pay without cards. Documentation
   What you can do:
   Generate Paycodes (unique codes for ATM/POS withdrawals)
   Create bulk Paycodes for multiple customers
   Cardless payments

6. CARD 360 SERVICE
   Create, manage, and control debit/prepaid cards. Documentation.
   What you can do:
   Create physical or virtual cards
   Block/unblock cards
   Change/retrieve PINs
   Link cards to accounts
   Check card balances
   Key APIs:
   Create Card API
   Block/Unblock Card API
   PIN Management APIs
   Card Balance APIs

7. TRANSACTION SEARCH
   Find and track payment transactions. Documentation.
   What you can do:
   Search transactions by reference
   Quick search across all transactions
   Bulk transaction queries
   Get detailed transaction information
   Key APIs:
   Quick Search API
   Reference Search API
   Bulk Search API
   Transaction Details API

8. IDENTITY VERIFICATION
   Verify customer identities (KYC - Know Your Customer). API + Documentation
   What you can do:
   Verify Bank Verification Number (BVN)
   Verify National Identity Number (NIN)
   Verify passport details
   Verify driver's license
   Verify voter's card and so much more
   Key APIs:
   BVN Verification API
   NIN Verification API
   Passport Verification API
   Driver's License Verification API
